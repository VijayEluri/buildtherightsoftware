Welcome :)

  1. Create a GreenPepper specification
  2. Share the link to it

To try a code for a spec:

  • Create a Maven project.
  • (the system will execute these maven goals : clean cobertura:cobertura)
  • (you can put your greenpepper fixtures in both the main or test maven's directories)
  • Share it on github.com or bitbucket.org
  • (the system will pull with git for github hosted projects and with mercurial for bitbucket hosted project)
  • Go to the spec you need to make pass
  • Put the url of your repository in the input field below the spec and click 'Try this code !'

Enjoy :)

Github sample

  • Given the following table for a spec about the mastermind
    • Rule for mastermind
    given the secret when player plays does he win ?
